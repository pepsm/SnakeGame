# SnakeGame

 Classic snake game on the terminal, made with C and ncurses.

### Dependencies
```
sudo apt-get install libncurses5-dev
```
### Controls
|      Keys      |    Actions     |
|     :---:      |     :---:      |
| **a**          |     left       |
| **w**          |      up        |
| **d**          |     right      |
| **s**          |      down      |
| **q**          | Quits the game at any time |

### Usage
```
$ make
$ ./snake
```
